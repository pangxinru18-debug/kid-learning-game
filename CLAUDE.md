# 快乐学习乐园 — 二年级益智学习网站

## 项目类型
纯前端单文件 HTML（D:\software\Claude\kid-learning-game\index.html）
PWA 支持：manifest.json + sw.js（离线缓存 + 添加到主屏幕）

## 功能模块
- 数学：加减法闯关、乘法口诀、混合运算、应用题
- 英语：看图识词、拼写练习、句子填空
- 作业模式：番茄钟（20分钟专注 + 3分钟休息）、三科任务跟踪、星级奖励、历史记录
- 游戏化：星星/连击/成就系统、Web Audio 音效

## 手机使用
1. 电脑上双击 `start-server.bat` 启动服务器
2. 手机连同一 WiFi，浏览器打开显示的地址
3. 手机浏览器菜单 → "添加到主屏幕"
4. 之后孩子点桌面图标即可打开（像普通 App）

## 测试方式
直接浏览器打开 index.html，或用 Live Server / start-server.bat。
测试设备优先级：手机 > iPad 平板 > 桌面浏览器。

## 修改原则
- 核心逻辑在 index.html 单文件中（HTML/CSS/JS）
- PWA 支持文件（manifest.json, sw.js）独立但最小化
- 无外部 CDN 依赖，保持零安装
- UI 面向 7-8 岁儿童：大按钮、大字体、色彩丰富
- 所有界面文字使用中文
